## Getting Started ğŸ

To view or contribute to this project locally, follow these steps:

1. Download the repository :
    ```
    git clone https://github.com/Creator-in/TidyMind.git
    ```

2. Navigate to the Project Folder :

    ```
    cd TidyMind
    ```
3. Install Packages :

    ```
    npm install
    ```

4. Create `.env` file inside the `TidyMind/` directory and add your environment variables:

    ```
    VITE_GEMINI_API_KEY= <YOUR_GEMINI_API_KEY>
    VITE_GEMINI_MODEL_URI = <YOUR_GEMINI_MODEL_URI>
    ```

5. Run the App :

    ```
    npm run dev
    ```

    The website is available at http://localhost:5173
